z-index is the CSS property which control stacking order of overlapping element

1 : Priority :- So here, we give highest (z-index:5;) to (BLUE COLOR BOX), so it is displaying first

2 : Priority :- Then we give (z-index:4;) to (GREEN COLOR BOX), so it will placed below the z-index number(5)

2 : Priority :- Then we give (z-index:2;) to (BLACK COLOR BOX), so it will placed below the z-index number(4)
